HTML Forms

form

The <form> element is a container for different types of input elements, such as: text fields, checkboxes, radio buttons, submit buttons, etc. It represents a document section containing interactive controls for submitting information.

An HTML Form is used to collect user input. The user input is most often sent to a server for processing.
