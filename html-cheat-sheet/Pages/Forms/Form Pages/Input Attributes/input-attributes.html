Input Attributes

<input>

The <input> attribute Tin HTML add features or behavior to an <input> element. Attribute examples include size, value, maxlength, required, and many more. It is one of the most powerful and complex in all of HTML due to the sheer number of combinations of input types and attributes.

The vast majority of the different input elements are covered in the list below:

  • <autofocus>
  • <disabled>
  • <form>
  • <heightandwidth>
  • <list>
  • <maxlength>
  • <minandmax>
  • <multiple>
  • <pattern>
  • <placeholder>
  • <readonly>
  • <required>
  • <size>
  • <step>
  • <type>
  • <value>
