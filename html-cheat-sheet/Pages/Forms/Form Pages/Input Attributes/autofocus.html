autofocus

The <autofocus> attribute tag is a boolean attribute. When present, it specifies that an <input> element should automatically get focus when the page loads.

Example, let the "First name" input field automatically get focus when the page loads: 

                    <form action="/action_page.php">
                    <label for="fname">First name:</label>
                    <input type="text" id="fname" name="fname" autofocus><br><br>
                    <label for="lname">Last name:</label>
                    <input type="text" id="lname" name="lname"><br><br>
                    <input type="submit">
                    </form>
