HTML <input> reset Attribute

reset

The reset defines a reset button which resets all form values to its initial values.

Tip: Avoid reset buttons in your forms! It is frustrating for users if they click them by mistake.

Example:

An HTML form with a reset input field: 

                    <form action="/action_page.php">
                    <label for="email">Enter your email:</label>
                    <input type="email" id="email" name="email"><br><br>

                    <label for="pin">Enter a PIN:</label>
                    <input type="text" id="pin" name="pin" maxlength="4"><br><br>

                    <input type="reset" value="Reset">
                    <input type="submit" value="Submit">
                    </form>

Display a Reset Button

Click on the reset button to reset the form.
