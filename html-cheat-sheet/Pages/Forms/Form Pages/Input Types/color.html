HTML <input> color Attribute

color

The <input type="color"> The defines a color picker. The default value is #000000 (black). The value must be in seven-character hexadecimal notation.

Tip: Always add the <label> tag for best accessibility practices!

Example:

Show a color picker (with a predefined red value): 

                    <form action="/action_page.php">
                    <label for="favcolor">Select your favorite color:</label>
                    <input type="color" id="favcolor" name="favcolor" value="#ff0000"><br><br>
                    <input type="submit">
                    </form>

Show a Color Picker



Note: type="color" is not supported in Internet Explorer 11 or Safari 9.1 (or earlier).
