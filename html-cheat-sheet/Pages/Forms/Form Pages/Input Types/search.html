HTML <input> search Attribute

search

The search defines a text field for entering a search string.

Note: Remember to set a name for the search field, otherwise nothing will be submitted. The most common name for search inputs is q.

Example:

An HTML form with a search input field: 

                    <form action="/action_page.php">
                    <label for="gsearch">Search Google:</label>
                    <input type="search" id="gsearch" name="gsearch">
                    <input type="submit">
                    </form>
