HTML <input> url Attribute

url

The url The <input type="url"> defines a field for entering a URL.

The input value is automatically validated before the form can be submitted.

Tip: Always add the <label> tag for best accessibility practices!

Example:

Define a field for entering a URL: 

                    <form action="/action_page.php">
                    <label for="homepage">Add your homepage:</label>
                    <input type="url" id="homepage" name="homepage">
                    <input type="submit">
                    </form>

Display a URL Input Field




Note: type="url" is not supported in Safari or Internet Explorer 12 and earlier versions.

Click on the submit button to submit the form.
