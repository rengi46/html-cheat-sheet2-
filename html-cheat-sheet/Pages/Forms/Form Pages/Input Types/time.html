HTML <input> time Attribute

time

The time The <input type="time"> defines a control for entering a time (no time zone).

Tip: Always add the <label> tag for best accessibility practices!

Example:

Define a single-line time field that a user can enter time into: 

                    <form action="/action_page.php">
                    <label for="fname">First name: </label>
                    <input type="time" id="fname" name="fname">
                    <label for="lname">Last name: </label>
                    <input type="time" id="lname" name="lname">
                    <input type="submit" value="Submit">
                    </form>

Show a Time Input Control

If the browser supports it, a time picker pops up when entering the input field.


Note: type="time" is not supported in Safari or Internet Explorer 12 and earlier versions.

Click on the submit button to submit the form.
