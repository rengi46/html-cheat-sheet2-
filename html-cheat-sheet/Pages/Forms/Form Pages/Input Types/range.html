HTML <input> range Attribute

range

The <input type="range"> defines a a control for entering a number whose exact value is not important (like a slider control). Default range is 0 to 100. However, you can set restrictions on what numbers are accepted with the attributes below. max - specifies the maximum value allowed min - specifies the minimum value allowed step - specifies the legal number intervals value - Specifies the default value

Tip: Always add the <label> tag for best accessibility practices!

Example:

Define a range control (like a slider control): 

                    <form action="/action_page.php">
                    <label for="vol">Volume (between 0 and 50):</label>
                    <input type="range" id="vol" name="vol" min="0" max="50">
                    <input type="submit">
                    </form>
